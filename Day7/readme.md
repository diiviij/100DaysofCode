Output Format

For each input variable  and appropriate primitive , you must determine if the given primitives are capable of storing it. If yes, then print:

Sample Input
5
-150
150000
1500000000
213333333333333333333333333333333333
-100000000000000

* short
* int
* long

* int
* long

* int
* long
